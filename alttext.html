What is Alt Text and Why It Matters

Alt text (alternative text) is a brief description of an image that screen readers announce to blind and visually impaired users. When written well, it allows these users to understand visual content and participate fully. When written poorly, it creates frustrating barriers.

Without Alt Text

A user hears: "Image, graphic, IMG_2847.jpg"

The meaning is lost.

With Good Alt Text

A user hears: "Bar chart showing sales increased 25% in Q2."

The meaning is clear.

The Data is Clear

Research consistently shows that effective alt text dramatically improves the user experience and task success.

40%

Reduction in task completion time for screen reader users with effective alt text.

60%

More likely to abandon tasks when alt text is missing or unhelpful.

Top Barrier

Missing or poor alt text is consistently ranked as a top accessibility problem by users.

Common Alt Text Mistakes

Avoid these common pitfalls that frustrate users and provide no value.

Mistake: Stating the Obvious

"Image of..." or "Picture of..."
(The screen reader already announces it's an image.)

Mistake: Being Too Vague

"Graph" or "Logo"
(What does the graph show? Which logo is it?)

Mistake: Focusing on Visuals

"Red button with rounded corners"
(Describe the function, not the CSS.)

Mistake: Using the Filename

"IMG_2847.jpg" or "hero-banner-v3.png"
(This provides no meaningful information.)

Writing Alt Text by Image Type

The right alt text depends entirely on the image's purpose. Here’s how to handle different types.

1. Informational Images (Charts, Graphs)

Bad: "Bar chart with colored bars"

Good: "Bar chart showing sales increased 25% from Q1 to Q2, reaching $2.3 million."

2. Functional Images (Buttons, Links)

Bad: "Magnifying glass icon"

Good: "Search for products"

3. Decorative Images

Adds no information, only visual flair.

Bad: "Abstract background with repeating cubes"

Good: alt="" (An empty alt attribute tells the screen reader to skip it.)

4. Logos and Branding

Acme Corp

Bad: "Logo"

Good: "Acme Corporation logo"

Technical Implementation

Here’s what the code looks like for different scenarios.

HTML Best Practices

<!-- ✅ Good: An informational image -->
<img src="sales-chart.png" alt="Q2 sales up 25% to $2.3M">

<!-- ✅ Good: A decorative image is silenced -->
<img src="decorative-border.png" alt="">

<!-- ✅ Good: An image used as a link -->
<a href="/products">
  <img src="new-product.jpg" alt="Shop our new summer collection">
</a>

<!-- ❌ Bad: Missing alt attribute -->
<img src="important.png">

<!-- ❌ Bad: Useless alt attribute -->
<img src="team-photo.jpg" alt="photo">

The Bottom Line

Effective alt text isn't about describing what you see—it's about conveying what's important for users to understand. It's a fundamental part of creating an inclusive and accessible web for everyone.

Think of alt text as translation: you're translating visual information into words that serve the same purpose.
